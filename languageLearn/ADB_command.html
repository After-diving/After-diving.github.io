ADB 命令

  1. adb shell 进入UE
  2. cat /proc/cpuinfo 打印cpu信息
  3. df -h 获取UE真实的存储空间
  4. wm size 获取UE的分辨率
  5. cat /system/bulid.prop 设备信息(修改?)
  6. getprop ro.bulid.version.release 系统版本
